# SCB 渣打银行战略合作知识库

## 1. SCB 定义

本项目中的 SCB 指 Standard Chartered Bank，即渣打银行。

本知识库用于沉淀 Dowsure 与渣打银行战略合作的历史沿革、合作目标、项目组合、推进机制、关键里程碑、风险卡点及后续行动计划，作为 SCB 战略合作项目管理的基础信息库。

## 2. 合作起点与背景

| 项目 | 内容 |
| --- | --- |
| 合作起点 | 2025 年 |
| 最早接触时间 | 2024 年 |
| 最初牵头人 / 团队 | Dowsure 创始人 Byron |
| 初始业务诉求 | 为跨境电商卖家打造匹配的信贷金融产品，并逐步延展至全方位金融服务解决方案 |
| 渣打银行看重 Dowsure 的能力 | 与亚马逊平台的合作基础，以及面向跨境电商卖家的数据能力 |
| Dowsure 看重渣打银行的资源 | 国际大行品牌影响力、全球金融服务能力，以及潜在股权投资机会 |
| 股权合作进展 | SCV（Standard Chartered Ventures）即将成为 Dowsure C+ 轮数百万美元级别的重要股权投资人 |

## 3. 战略合作主线

当前与渣打银行的合作可以初步归纳为三条主线：

1. Dowalet eCAS 账户体系搭建
2. 跨境电商企业及企业主开户推荐与 Digital Onboarding 方案探索
3. Project Dscale 大额跨境多币种信贷产品合作

## 3.1 战略合作北极星目标

Dowsure 与 SCB 合作的最终战略目标，是成为全球战略伙伴，共同打造“SCB + Dowsure”的跨境电商金融服务标杆，为跨境电商生态企业，并逐步延展至更广义的跨境贸易企业，提供优质、完整、可规模化的金融服务解决方案。

在这一过程中，Dowsure 希望通过 SCB 的国际大行背书扩大自身品牌影响力，并借助 SCB 的银行能力、账户能力、风控与合规资源，持续完善 Dowalet 及相关金融基础设施。

## 3.3 当前合作属性判断

当前 Dowsure 与 SCB 的合作属于复合型战略合作，覆盖资本合作、银行产品合作、跨境电商生态合作、Dowalet 基础设施合作、品牌与市场合作等多个方向。后续管理重点不是简单归类，而是明确主次、阶段优先级、资源投入及项目组合治理机制。

## 3.5 SCB 战略合作管理项目自身目标

本项目并非单纯整理 SCB 合作档案，而是以 SCB 为样板，建设“战略资方管理 AI 系统”的 MVP。

该 MVP 的目标是搭建知识库、任务管理、客户进度管理、仪表盘和自动汇报能力，以解决 SCB 合作中信息分散、人工盯办、跨部门协同成本高、会议事项遗漏、业务数据缺少统一看板等问题。

需要注意的是，“SCB 战略合作管理系统 MVP”不是 SCB 合作下的一条业务项目线，而是战略资方合作管理的基础设施。未来该系统应能对接不同战略合作资方的知识库 / 数据库，并根据各资方实际合作内容，智能生成项目线、管理模块、数据看板、日报 / 周报 / 月报和规划文档。

以 SCB 为例，第一版业务项目线包括：

- eCAS 账户体系。
- Project Dscale。
- 企业开户 / Digital Onboarding。
- 企业主 / 高管个人开户。
- MFP 长期合作。

市场活动及品牌共建不放在项目组合看板中作为普通业务项目线管理，而是作为系统左侧一级工作模块，与项目组合、客户推进、知识库、项目管理档案库等模块并列。0714 SCB GBA Summit & Project Dscale PR 是该模块下的一次具体市场活动事项。物料库作为“市场活动及品牌共建”模块下的子菜单 / 子标签页，不作为左侧一级菜单。

SCV 投资协同暂作为战略背景、重大里程碑和资本资源投入观察项，不进入 SCB 日常项目组合看板，应归档到项目管理档案库。

Referral Agreement 等协议规则不作为项目组合看板中的子项目展示，应归入项目管理档案库，并作为知识库问答、汇报生成和收入规则解释的底层资料。

第一版系统中，SCB 项目线管理深度如下：

深度管理：
- eCAS 账户体系。
- Project Dscale。

轻量管理：
- 企业开户 / Digital Onboarding。
- 企业主 / 高管个人开户。
- MFP 长期合作。

独立深度工作模块：
- 市场活动及品牌共建。

## 3.4 当前项目管理痛点

- 合作业务线多，信息分散，缺少统一知识库进行沉淀与自动化梳理。
- 推进高度依赖人工盯办，跨部门沟通费时费力。
- 会议后行动事项容易遗漏，缺少自动化、智能化的任务分拆、提醒和进度管理机制。
- 缺少统一仪表盘追踪业务数据，导致总结汇报耗时。
- 后续需继续补充更多痛点，并转化为系统功能需求和管理流程设计。

## 4. 项目一：Dowalet eCAS 账户体系搭建

### 项目定位

Dowalet eCAS 是 Dowsure 与渣打银行合作的底层账户体系，支撑跨境电商卖家的资金结算、信贷放款及支付等场景。当前阶段优先保障卖家收款及还款场景落地。

### 当前进展

- Dowsure 已提交全部合规及审批文档。
- SCB 暂未要求补充材料。
- 项目已由 SCB SME 团队转至 FI 团队推进。
- 预计 2026 年 6 月底前上线。
- 当前核心卡点为 SCB 内部审批流程及技术联调。
- SCB 团队初步预估实施周期为 1.5 个月，Dowsure 正推动压缩至 1 个月内完成。

### 功能设计

账户结构：
- 采用“主账户 + 子账户”模式。
- 子账户冻结指令由 Dowsure 控制。
- 支持多币种独立子账户设置。
- 账户名称支持大陆公司拼音名。

资金管理：
- EOD 资金自动扫回主账户。
- 子账户 T+1 全量回款。
- 客户自主入金需明确出金指令，否则原路退回。
- Payout 场景下需同步发送客户付款指令及 Dowsure 佣金收取指令。

合规要求：
- 无 MSO 牌照阶段，仅支持收款及还款路径。
- 取得 MSO 牌照后，可拓展放款及资金留存场景。
- Olea 资金需直接授信至商家 eCAS 账户，当前正推进 API 集成可行性。

长期规划：
- Phase 1 仅准入卖家。
- Phase 2 纳入服务商等生态角色。
- 长期目标是在 eCAS 内实现较高比例的资金留存。

## 6. 项目三：Project Dscale 大额跨境多币种信贷产品合作

### 产品定位

Project Dscale 面向跨境电商卖家，提供“信贷 + 账户 + 结算”综合解决方案。

### 初步产品参数

| 维度 | 内容 |
| --- | --- |
| 授信额度 | 200-500 万美元 |
| 支持币种 | 美元 / 人民币 |
| 额度类型 | 循环额度 |
| 单笔支用期限 | 3-6 个月以上 |
| 利率 | 7-10%，最低可至 6%-6.5%；具体口径待确认 |

### 准入标准

## 8. 其他合作方向

以下信息来自飞书知识问答摘要，待后续结合原始资料逐条核验。

### MFP 合作

- MFP 属于长期目标，现阶段暂未进入实际推进阶段，当前主要停留在合作提案层面。
- 初步设想是以 MFP 作为长期引流入口，由 SCV 投资的 Olea 作为主要资方，引导客户开立 eCAS 子账户，再交叉营销企业开户、Dscale、企业主 / 高管个人开户等产品。
- MFP 产品合作方案中提出，以 Fastpay / 提前收款服务为入口，为卖家提供信贷支持，同时引导客户将结算业务转移至 SCB 账户；资金沉淀后，银行侧可进一步通过 FX、存款、理财等业务形成收益。
- 对 Dowsure 的战略价值在于：将信贷、结算、账户、客户经营串联起来，长期形成“信贷 + 结算 + 增值服务”的跨境电商金融闭环。

### Affluent 财富管理

- Affluent 财富管理不作为单独合作主线管理，当前归入“企业主 / 高管个人开户”相关场景。
- 已累计推荐 5 位企业主及高管开户，其中 4 位已落地。
- 后续如推进 Digital Onboarding 二维码嵌入 Dowalet 卖家个人页，可作为企业主 / 高管个人开户和财富管理转化的延伸能力。
