# SCB 战略资源投入评分模型

## 1. 当前优先级判断

根据当前业务判断，SCB 对 Dowsure 的战略资源投入优先关注以下四类：

| 优先级 | 资源类型 | 关注原因 |
| --- | --- | --- |
| P1 | 产品资源 | 直接影响 Dscale、eCAS、开户等产品能否形成差异化竞争力和规模化转化 |
| P2 | 组织资源 | 直接影响 SCB 内部推进效率、跨团队协同、审批速度和问题闭环 |
| P3 | 资本资源 | SCV 投资是战略合作升级的重要背书，也可能带来集团级协同 |
| P4 | 品牌资源 | 联合发布、GBA Summit、市场背书有助于提升客户信任和行业影响力 |

补充判断：
- CEO Byron 对资本资源高度关注，尤其是 SCV 投资对公司融资和集团级背书的价值。
- 但从 SCB 战略合作管理系统的日常使用视角看，产品资源和组织资源更直接影响业务推进效率、客户转化和收入结果。
- 资本资源主要由 CM / Capital Market 团队重点跟进，本系统应将其作为战略背景、重大里程碑和资源协同信号，而非日常高频盯办指标。

暂不作为第一优先级但可观察的资源：
- 客户资源：SCB 是否向 Dowsure 推荐客户或开放客户网络。
- 技术资源：API、Digital Onboarding、eCAS 对接、数据互通。
- 风险 / 合规资源：审批支持、合规解释、牌照路径、结构设计。

技术资源和风险 / 合规资源虽然重要，但当前可先归入“产品资源”和“组织资源”的二级指标中管理。

## 2. 评分方式

建议采用 1-5 分评分：

| 分数 | 含义 |
| --- | --- |
| 1 分 | 仅口头表达支持，暂无实际投入 |
| 2 分 | 有初步讨论或个别支持动作，但尚未形成机制 |
| 3 分 | 已形成明确方案、owner 或阶段性动作，能支持单点项目推进 |
| 4 分 | 已形成稳定机制，并能持续支持多个客户或多个项目 |
| 5 分 | 已成为双方战略级资源配置，具备可复制、可规模化、可对外背书的合作机制 |

## 3. 一级评分维度

### 3.1 产品资源

衡量 SCB 是否为 Dowsure 合作提供差异化产品能力。

可观察信号：
- 为 Dowsure 客户设计专属或半专属产品方案。
- 对 Dscale 提供有竞争力的定价、额度、期限或币种支持。
- 定期更新白名单企业额度。
- 对优质客户提供快速评审通道和加急审批选项。
- 允许基于淡旺季、业绩表现灵活调整额度。
- 支持材料线上电子预审、审批进度透明化。
- 将 eCAS、Dscale、开户、MFP 等产品打包成综合解决方案。

0424 workshop 已出现的正向信号：
- 提出对齐预授信参考区间。
- 提出定期更新白名单企业额度。
- 提出快速评审通道、超时处理机制、3 周流程时效、首次放款与续贷审批区分、加急审批选项。
- 提出共建跨境电商数字化金融服务平台。

### 3.2 组织资源

衡量 SCB 是否投入足够组织资源推动合作闭环。

可观察信号：
- 有 SCB 高层 sponsor 参与，包括 Global / Regional / China / HK 相关负责人。
- FI、SME、Trade Finance、RM、风险、合规、技术等团队形成协同机制。
- 明确各环节负责人、审批时效和合规责任边界。
- 建立专属合作团队。
- 建立定期会议、问题闭环跟踪体系和升级机制。
- 对 POC、Phase 1、Phase 2 设置明确时间表和责任团队。

0424 workshop 已出现的正向信号：
- 参会层级包括 SCB SME Global Head。
- 提出明确任务分工及合规责任边界。
- 提出建立专属合作团队、定期业务沟通会议和问题闭环跟踪体系。
- 设置 POC、Phase 1、Phase 2 的阶段时间表。

### 3.3 资本资源

衡量 SCB / SCV 是否通过资本或资本相关协同支持 Dowsure。

可观察信号：
- SCV 投资金额、轮次、签约和交割进度。
- SCV 投后是否推动 SCB 内部资源协同。
- 投资是否带来集团级背书、跨区域资源或业务 sponsor。
- 资本合作是否与 eCAS、Dscale、MFP、Olea 等业务线形成联动。

当前已知正向信号：
- SCV 即将成为 Dowsure C+ 轮数百万美元级别的重要股权投资人。
- Olea 作为 SCV 投资相关主体，可作为 MFP 长期目标中的潜在主要资方。

### 3.4 品牌资源

衡量 SCB 是否愿意与 Dowsure 共同对外建立行业标杆影响力。

可观察信号：
- 联合举办或联合露出活动。
- SCB GBA Summit 等高规格活动中展示 Dowsure 合作。
- Project Dscale 联合官宣。
- SCB 对外背书 Dowsure 的跨境电商金融能力。
- 媒体、行业会议、白皮书、客户案例等联合传播。

当前已知正向信号：
- 计划在 2026 年 7 月 14 日 SCB GBA Summit 上合并举办 Project Dscale 产品官宣活动。
- Dowsure 计划邀约 30-40 家年 GMV 超 1 亿元跨境电商企业参与。

## 4. 建议仪表盘字段

| 字段 | 示例 |
| --- | --- |
| 资源类型 | 产品 / 组织 / 资本 / 品牌 |
| 当前评分 | 1-5 |
| 上期评分 | 1-5 |
| 变化原因 | 新增高层 sponsor、审批机制落地、活动确认等 |
| 关键证据 | 会议纪要、邮件、合同、活动方案、产品方案 |
| 负责人 | Dowsure owner + SCB owner |
| 下一步动作 | 待确认或待推动事项 |
| 风险 / 卡点 | 审批慢、owner 不清、资源未兑现等 |

## 5. 建议权重

建议采用“日常经营型权重”，突出产品和组织资源：

| 维度 | 建议权重 | 管理含义 |
| --- | --- | --- |
| 产品资源 | 40% | 是否真正形成有竞争力、可落地、可规模化的产品能力 |
| 组织资源 | 35% | 是否有足够 SCB 内部资源、owner、机制和时效保障项目推进 |
| 品牌资源 | 15% | 是否通过联合活动和对外背书形成市场影响力 |
| 资本资源 | 10% | 作为战略背景和重大里程碑观察，不作为日常高频管理重点 |

如需向 CEO / 董事会汇报，可另行生成“CEO 视角权重”：

| 维度 | CEO 视角参考权重 |
| --- | --- |
| 产品资源 | 30% |
| 组织资源 | 25% |
| 资本资源 | 30% |
| 品牌资源 | 15% |

本系统默认采用日常经营型权重；在资本合作出现重大节点时，再切换或补充 CEO 视角。

## 6. 待确认问题

- 技术资源是否需要从产品资源中拆出为独立维度，尤其是 eCAS 和 Digital Onboarding 推进进入实施期后？
