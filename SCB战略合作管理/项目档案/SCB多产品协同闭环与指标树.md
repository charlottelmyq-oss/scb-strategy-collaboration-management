# SCB 多产品协同闭环与指标树

## 1. 产品 / 服务范围

### 当前应纳入协同闭环的产品 / 服务

| 产品 / 服务 | 当前定位 | 管理状态 |
| --- | --- | --- |
| eCAS | 底层账户基础设施，支撑资金结算、还款、后续放款及资金留存场景 | 2026 年目标一，重点落地 |
| Project Dscale | 大额跨境多币种信贷产品，面向头部及高质量跨境电商卖家 | 2026 年目标二，重点规模化 |
| 企业开户 | 企业客户进入 SCB 体系的基础入口之一 | POC 至标准化推进 |
| 企业主 / 高管个人开户 | 包含 Affluent 财富管理相关转化场景 | 与企业开户及客户关系经营联动 |
| Digital Onboarding | 开户规模化、线上化、低摩擦转化能力 | 分阶段探索 |
| MFP | 长期目标，作为未来流量入口和“信贷 + 结算 + 增值服务”闭环的一部分 | 当前停留在合作提案阶段，暂未实际推进 |

### 不单独作为主线管理的内容

- Affluent 财富管理：纳入“企业主 / 高管个人开户”场景，不单独作为当前核心管理主线。
- AUM / 财富管理转化：属于其他部门重点关注指标，当前不进入本项目核心指标。
- eCAS 留存资金：属于其他部门重点关注指标，当前不进入本项目核心指标，但可作为观察指标保留。

## 2. 理想客户转化路径

长期理想路径：

```text
MFP / Fastpay / Olea 资金产品
  -> 引导客户开立 eCAS 子账户
  -> 基于 eCAS 与客户数据识别需求
  -> 交叉营销企业开户
  -> 交叉营销 Project Dscale
  -> 交叉营销企业主 / 高管个人开户
  -> 对少量重点 Dscale portfolio 客户进行生命周期价值经营
```

当前阶段可执行路径：

```text
Project Dscale 重点客户 / SCB 开户 POC 客户 / GBA Summit 高质量客户
  -> 完成企业开户或开户意向确认
  -> 匹配 Dscale 授信需求
  -> 推动 eCAS 账户体系承接收还款或后续资金路径
  -> 推荐企业主 / 高管个人开户
  -> 沉淀客户进度、转化结果与收入贡献
```

## 3. MFP 长期闭环定位

MFP 现阶段不是执行主线，而是长期协同闭环中的潜在前端入口。

根据 MFP 产品合作方案，长期模式可以概括为：

- 以 Fastpay / 提前收款服务作为核心入口。
- 由 Olea 等资方提供资金支持，服务跨境电商卖家资金周转需求。
- 引导客户将结算业务迁移至 SCB 账户或 eCAS 子账户。
- 后续通过 FX、存款、理财、企业开户、Dscale 等产品形成多产品转化。
- 对 Dowsure 来说，MFP 的战略价值是扩大客户入口和交易场景；对 SCB 来说，价值是获取账户、结算、FX、存款及财富管理等银行业务机会。

## 4. 核心指标树

### 4.1 当前核心关注指标

| 指标 | 说明 | 归属目标 |
| --- | --- | --- |
| 资产池规模 | Project Dscale 及相关 SCB portfolio 的累计资产规模，2026 年目标超过 100M USD | Dscale 规模化 |
| 放款利差 / 放款收入 | 来自 Dscale 等信贷产品的收益贡献 | 商业收入 |
| 开户 referral fee 收入 | 企业开户、企业主 / 高管开户等推荐带来的 referral fee | 开户转化 |
| SCB 对 Dowsure 的战略资源投入 | 包括品牌背书、SCV 投资、银行资源、活动资源、产品资源、内部协同资源 | 战略合作深度 |
| 少量 Dscale portfolio 客户生命周期价值 | 针对重点客户观察多产品转化、持续合作与综合价值 | 客户经营 |

### 4.2 当前非核心但可观察指标

| 指标 | 当前处理方式 |
| --- | --- |
| AUM / 财富管理转化 | 由其他部门重点关注，本项目暂不作为核心指标 |
| eCAS 留存资金 | 由其他部门重点关注，本项目可作为观察指标保留 |
| FX 收益 | 对 SCB 侧价值重要，但 Dowsure 当前是否直接收益待确认 |
| 存款 / 理财收益 | 对 SCB 侧价值重要，当前不作为 Dowsure 核心收入指标 |

## 5. 仪表盘建议

第一版仪表盘建议分为五块：

| 看板 | 核心问题 | 关键指标 |
| --- | --- | --- |
| Dscale 资产池 | 资产规模是否按目标增长？ | 累计授信、累计放款、在贷余额、资产池规模、客户数 |
| 收入贡献 | Dowsure 从 SCB 合作中获得了什么收入？ | 放款利差 / 收入、referral fee、预计收入、已确认收入 |
| 开户转化 | 客户是否进入 SCB 账户体系？ | 企业开户数、企业主 / 高管开户数、开户成功率、开户阶段分布 |
| 战略资源 | SCB 是否投入了足够战略资源？ | SCV 投资进展、联合活动、品牌露出、SCB 内部资源协调、产品资源投入 |
| 重点客户 LTV | 少量重点客户是否形成多产品转化？ | 客户产品覆盖数、授信 / 放款、开户状态、后续产品机会、综合价值 |
